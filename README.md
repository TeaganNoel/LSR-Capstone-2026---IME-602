This repository contains data manipulation and analysis scripts for the corvallis land speed racing team.
The repository was created by Teagan Noel and Mason Allen, who are participating members of the land speed racing team.
The purpose of this repository is to share and store python scripts created to assist the LSR team in their mission.
This project represents Teagan, Mason, and Rudy's efforts which are characterized as a senior capstone project.
The capstone project holds the title of "Testing Processes for Land Speed Racing" with a timeline of January 2026 - June 2026.
The capstone subteam holds the title of "IME 602".
